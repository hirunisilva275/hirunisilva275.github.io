Projects
    Analog Time delays
    In this work we realize analog memory using TTD elements for radar, communications, and AI/ML applications. The circuits were designed in TSMC 180nm CMOS technology and simulated at the transistor level using Cadence Spectre.
    Analog CMOS for computing non-linear Transcendental Functions
    Circuits for sin x(t), cos x(t), ln x(t) and exp(x(t)) based on truncated Taylor and Maclaurin expansion functions are designed and simulated using Cadence Spectre in the 180 nm CMOS
